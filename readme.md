# Telegram Sticker Auto-Delete Bot

A simple Telegram bot that deletes stickers after a set delay.

## Features
- Automatically deletes stickers after X seconds
- Works in groups and private chats
- Built with Telegraf (Node.js)

## Setup

1. Clone repo
2. Install dependencies:
   npm install

3. Create `.env` file:
   BOT_TOKEN=your_token_here

4. Run bot:
   node index.js

## Requirements
- Node.js
- Telegram bot token from @BotFather
